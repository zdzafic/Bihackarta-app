# Bihackarta 🗺️

Interaktivna karta Bihaća — web aplikacija za prikaz lokacija u gradu Bihaću.

## Pokretanje projekta

```bash
python3 -m http.server 8000
```

Zatim otvori u browseru: [http://localhost:8000](http://localhost:8000)

## Struktura projekta

```
Bihackarta-app/
├── index.html          # Glavna stranica
├── style.css           # Stilovi
├── script.js           # Logika i lokacije
├── README.md           # Ova datoteka
├── .gitignore
├── assets/
│   ├── images/         # Slike
│   └── icons/          # Ikone
├── data/
│   └── locations.json  # Podaci o lokacijama
└── pages/              # Extra pagina's (later)
```

## Tehnologije

- HTML5
- CSS3
- Vanilla JavaScript
